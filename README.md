# 🧠 Quizzly (Assam Edition)

A simple, clean, and interactive quiz application built using **HTML, CSS, and JavaScript**.
This project focuses on core frontend concepts like DOM manipulation, event handling, and state management — without any external libraries.

---

## 🚀 Features

* ✅ Multiple choice questions
* 🎯 Score tracking system
* 🔄 Restart quiz functionality
* 🎨 Clean and modern UI
* ⚡ Smooth question transitions
* 🌐 Fully client-side (no backend required)

---

## 📚 Quiz Topic

This version includes **10 questions based on Assam**, covering:

* Geography
* Culture
* Festivals
* Important landmarks

---

## 🛠️ Tech Stack

* **HTML** → Structure
* **CSS** → Styling & UI
* **JavaScript** → Logic & interactivity

---

quiz-app/

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start answering questions 🎉
